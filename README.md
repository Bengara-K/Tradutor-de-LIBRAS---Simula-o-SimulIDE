O arquivo luvaTESTE.sim1 é o arquivo da esquemática dentro do SimulIDE

Os arquivos .hex são compilações hexadecimais dos arquivos .ino
O formato .hex é utilizado como firmware nos microcontroladores do simulador SimulIDE
